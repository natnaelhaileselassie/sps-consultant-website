# Requirements
Below describes current functional and nonfunctional requirements.

# Functional
The system shall…
- display information regarding the client
- have a working contact page which allows users to send a message on the page which reaches client’s email
- have a secure login to access website as an admin with elevated privileges 
- have a section which allows users to access useful links
- display only the following until further notice: Home, Services, About, Contact, Useful Links
- have the following pages: Home, Services, Portfolio, About, Contact, Partners, Useful Links

# Nonfunctional
The system shall…
- be easy for the client to add content without the developer
- load under 2 seconds
- have hidden pages which can be updated and later easily revealed
- be available to be reached in Ethiopia and Internationally around the world
